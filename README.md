# LX Music Mobile

LX Music Mobile 当前分支，主要聚焦 iOS 适配与移动端可用性补齐。

## 项目说明

- 基于 React Native 开发
- 当前仓库说明仅针对这个分支，不代表上游官方发布内容
- 上游仓库：<https://github.com/lyswhut/lx-music-mobile>
- 桌面版项目：<https://github.com/lyswhut/lx-music-desktop>

## 当前状态

当前版本已完成一轮较完整的 iOS 可用性适配，日常播放相关功能基本可用，包括：

- 应用图标、名称与基础资源适配
- 音乐播放、进度同步、时长获取
- 后台播放
- 锁屏 / 控制中心信息展示与基础控制
- 在线源播放
- 自定义源 / User API
- 歌曲封面显示
- 评论、换源、榜单

## 平台支持

- Android 5+
- iOS

## 已知限制

- 暂不支持桌面歌词
- 首次启动若显示英文，可在设置中切换为中文
